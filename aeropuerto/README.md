# Sistema de Reservas de Aeropuerto - Emirates

Sistema web desarrollado en Angular para la gestión de reservas de vuelos y administración de aeropuerto.

## Descripción del Proyecto

Aplicación web que permite a los usuarios registrarse, iniciar sesión, buscar vuelos y realizar reservas. Incluye un panel de administración para la gestión del sistema.

## Características Principales

- Sistema de autenticación (Login/Registro)
- Registro de usuarios con validación de formularios
- Panel de administración (Dashboard)
- Carrito de compras para reservas
- Sistema de rutas protegidas con Guards
- Interfaz responsive con Bootstrap 5
- Diseño basado en Emirates Airlines

## Tecnologías Utilizadas

- **Framework:** Angular 15
- **Lenguaje:** TypeScript
- **Estilos:** CSS + Bootstrap 5
- **Control de versiones:** GitHub
- **IDE:** Visual Studio Code
